盒模型

在标准模式下，一个块的总宽度= width + padding(左右) + border(左右) + margin(左右)

而IE盒模型或怪异盒模型显而易见的区别就是，width和height除了content区域外，还包含padding和border。盒子的总宽度为

一个块的总宽度= width + margin(左右)（即width已经包含了padding和border值）
